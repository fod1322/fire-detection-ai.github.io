# 🔥 Система обнаружения пожара

Проект на Python, использующий искусственный интеллект для определения пожара на изображениях.

## 🚀 Возможности
- Загрузка изображения
- Определение наличия пожара
- Графический интерфейс (Tkinter)

## ▶️ Как запустить

1. Установить зависимости:
pip install -r requirements.txt

2. Запустить программу:
python main.py
